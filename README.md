 2. maven
 
##### 技术栈

### 目录结构
<pre>
.
├── base                   // 基础文件
│	├── aop                // 
│   ├── bean               // 基本bean类
│   ├── config             //数据源配置
│   ├── filter              // 过滤器
│	└── exception          // 异常处理文件
│         
├── constants                   // 常量
├── controller               // api接口
├── dao                      // 数据层
├── entity                   // 实体类
├── mapper                   // 数据存储对象
├── utils                    // 工具类
├── service                  // 业务逻辑处理层
├── .gitignore               //git忽略文件
</pre>
